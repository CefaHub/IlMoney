# IllitMoney

IllitMoney — плагин валюты «айлитики» для Paper/Spigot 1.20.1.  
Плагин хранит баланс игроков, поддерживает переводы, топ по балансу и PlaceholderAPI.  
Также он предоставляет экономику через Vault API (как провайдер), чтобы другие плагины (например IllitAuction) могли работать с валютой без правок.

---

## Команды

### Игроки
- `/balance` (`/bal`) — показать баланс
- `/pay <ник> <сумма>` — перевести айлитики игроку
- `/baltop` — топ по балансу

### Админ-команды (требуется `illit.money`)
Все админ-команды выполняются через `illit`:

- `/illit money give <ник> <сумма>` — выдать айлитики
- `/illit money take <ник> <сумма>` — снять айлитики
- `/illit money set <ник> <сумма>` — установить баланс
- `/illit money reload` — перезагрузить конфиг/сообщения (если включено в сборке)

---

## PlaceholderAPI плейсхолдеры

### Баланс
- `%Illit_balance%` — баланс игрока
- `%Illit_balance_formatted%` — форматированный баланс

### Топ (1–10)
- `%Illit_baltop_name_1%` … `%Illit_baltop_name_10%` — ник в топе
- `%Illit_baltop_1%` … `%Illit_baltop_10%` — баланс в топе

---

## Админ-команды и пермишены

Админ-доступ ко всем админ-командам:
- `illit.money`

Пример выдачи (LuckPerms):
- `lp group admin permission set illit.money true`
